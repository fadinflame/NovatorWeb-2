# NovatorWeb-2
Два скрипта для имитации создания и отправки писем с использованием PostgreSQL

## Использование
1. Вписать данные для подключения к бд в config.py

2. Запустить creator.py и executor.py

## Зависимости
- Тестировалось на python 3.7.0
- psycopg2 2.8.5
